# Roadmap

План развития проекта WinFlow.

## v0.3.0 — Возвращаемые значения и модули (Q1 2025)

### Возвращаемые значения функций
- [ ] Синтаксис `return value=...` в функциях
- [ ] Присвоение результата: `env set result=$(funcname(args))`
- [ ] Поддержка множественных возвратов
- [ ] Тесты рекурсивных функций

### Модули и импорты
- [ ] Система модулей: `import module.wflow`
- [ ] Пространства имен для функций
- [ ] Стандартная библиотека модулей
- [ ] Package manager (опционально)

### Улучшение обработки ошибок
- [ ] Типизированные исключения
- [ ] Stack trace для функций
- [ ] `finally` блоки
- [ ] Логирование ошибок в файл

- [ ] Словари/хеш-таблицы

## Приоритеты

Приоритеты определяются:
1. **Пользовательские запросы** — наиболее востребованные возможности
2. **Критические баги** — блокирующие проблемы
3. **Архитектурные улучшения** — фундамент для будущих возможностей
4. **Экосистема** — интеграции и инструменты
