Лаконичный синтаксис

Читабельные сценарии: команды и аргументы key=value, поддержка кавычек.

Полезные модули

Встроенные env, file, process; планируется registry.

CLI и Инсталлятор

Запуск из консоли и быстрый установщик без прав администратора.

Интеграция с Windows

Ассоциация .wflow, работа с переменными окружения и процессами.

CI/CD

Готовые workflows: сборка, релизы, синхронизация Wiki и документации.

Открытый код

.NET 8, чистая архитектура: Core, CLI, Installer, ShellHost.

Пример скрипта

#/// WinFlow Demo
echo message="Task 1: Environment setup"
env set name=APP_NAME value="WinFlow"
env print

echo message="Task 2: File ops"
file write path="config.txt" content="APP_NAME=WinFlow"

echo message="Task 3: Process"
process.exec file="cmd.exe" args="/c echo Hello"

echo message="Complete!"

Запуск: winflow demo.wflow --verbose
